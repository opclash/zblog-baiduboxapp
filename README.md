<h2>小程序特色</h2>
<li>集成 swan-sitemap（使用zblog API）</li>
<li>完善 swan.setPageInfo</li>
<li>添加 一站式互动组件</li>
<li>添加 一键关注</li>
<li>添加 在线客服</li>
<li>引用 mp-html 组件</li>
<li>文章页详情页支持 pre 代码展示</li>
<li>支持 视频、音频播放</li>
<li>拥有 首页、列表页、独立页</li>

<h2>依赖系统插件</h2>

插件：https://github.com/ossso/os_wxapp_one

需要修改部分PHP代码，需要懂nmp、php，修改请自行解决
