<h2>小程序特色</h2>
<li>使用 Zblog 1.7.1 原生API接口</li>
<li>小程序集成 swan-sitemap 索引页</li>
<li>文章页集成 swan.setPageInfo </li>
<li>添加 一站式互动组件</li>
<li>添加 一键关注</li>
<li>引用 mp-html 组件</li>
<li>支持 视频、音频播放</li>
<li>拥有 首页、列表页、独立页</li>

<h2>小程序依赖</h2>
<li>依赖Z-Blog 1.7.1 的API接口，无需任何插件</li>
<li>打开后台->网站设置->API设置，打开 启用API协议开关 并提交即可</li>
